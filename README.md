# Bank Transactions — Data Cleaning

Data cleaning and exploratory analysis project on bank transaction records.

## Overview

This project applies a structured data cleaning methodology to raw bank transaction data, preparing it for downstream analytics and reporting.

## Tech Stack

**Python** · **Pandas** · **Jupyter Notebook**

## Project Structure

```
├── code/              # Cleaning scripts and notebooks
├── data/              # Raw and cleaned transaction data
├── documentation/     # Methodology and decisions
├── DECISIONS.md       # Data quality decision log
└── README.md
```

## Cleaning Steps

1. Duplicate detection and removal
2. Missing value handling strategy
3. Date and amount type normalization
4. Category standardization
5. Anomaly detection (outliers, negative balances)
6. Final dataset validation

## Key Output

Clean, analysis-ready transaction dataset with documented transformation decisions.
